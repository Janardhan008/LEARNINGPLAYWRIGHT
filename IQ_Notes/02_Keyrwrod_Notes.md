# JavaScript Keywords (Reserved Words)

## Explanation

**Keywords** (also called **reserved words**) are words that JavaScript uses for its own syntax and internal operations. They cannot be used as variable names, function names, class names, or identifiers in your code — doing so will cause a syntax error.

Some keywords are always reserved; others (marked with `*`) were added in ES6 (2015) or later. A separate set was removed from the ECMAScript 5/6 standard but is still widely avoided.
